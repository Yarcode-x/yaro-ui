Sistema de diseño YARO
Este documento es la única fuente de verdad del sistema de diseño. Todos los componentes de @yaro/ui referencian estos tokens. Nunca hardcodear colores, sombras o tipografía en los componentes.

Colores de superficie
Jerarquía de fondos del sistema. Se usan en capas: bg es el más profundo, white es el más elevado. Las cards siempre van sobre bg o bg2.
⚠ Nunca usar colores en hex directamente en componentes — siempre var(--color-*). Haz clic en cualquier token para copiarlo.
Superficies y texto
Acento
El verde-limón cálido es el color identificador de YARO. Tiene cuatro variantes según el uso: fondo completo, borde/barra, texto sobre fondo acento, y fondo sutil.
Colores de estado
Cada estado tiene tres variantes: base (texto/íconos), bg (fondos semitransparentes) y border (bordes de badges y alertas). El sistema es consistente — verde siempre es OK, rojo siempre es error. Haz clic en cualquier variante para copiar su token.
Sombras
Tres niveles de elevación crean la jerarquía visual de profundidad. El elemento con mayor sombra percibe estar más cerca del usuario. Haz clic para copiar.
Radios de borde
Tres niveles de redondez. Los valores más grandes van en los contenedores principales; los más pequeños en elementos dentro de ellos.
Tipografía
YARO usa exclusivamente Outfit en todos sus pesos. No mezclar con otras fuentes. JetBrains Mono solo para valores numéricos de precisión, código y YARO Console.
Muestra de la familia — Outfit
Light Regular Medium Semibold Bold
ABCDEFGHIJKLMNOPQRSTUVWXYZ · 0123456789 · $#@&
Espaciado
Escala de 4px. Todos los márgenes, paddings y gaps deben usar estos tokens. Nunca espaciados arbitrarios como 7px o 15px. Haz clic para copiar.
Transiciones
Pasa el cursor sobre cada card para ver la animación en acción. El spring es el más característico de YARO — se usa en la aparición de modales y cards.
Tema oscuro — YARO Console
Los mismos tokens CSS, valores distintos. La Console aplica [data-theme="console"] en el elemento raíz para sobrescribir todas las variables automáticamente.
Paleta Console — dark mode
[data-theme="console"] {
  --color-bg: #0a0a0b;
  --color-accent: #7c6af7;
  --color-text: #e8e8ec;
  /* ... todas las variables sobrescritas */
}
