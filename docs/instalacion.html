Instalación

@yaro/ui es una librería de componentes Angular standalone dentro del monorepo Nx. No se publica en npm — se consume directamente vía path alias desde el workspace. Sigue estos 4 pasos y tendrás acceso a todos los componentes.

ℹ
Requisitos: Angular 17+ · Nx workspace · Node 18+. La librería usa @if / @for de la nueva sintaxis de control flow — no requiere CommonModule.
Configuración inicial
Cuatro pasos, todos en archivos de configuración existentes. No es necesario instalar nada nuevo.
1
Path alias en tsconfig.base.json
Registra el alias @yaro/ui para que TypeScript sepa resolver los imports. El alias apunta al barrel index.ts de la librería.
tsconfig.base.json
{
  "compilerOptions": {
    "paths": {
      "@yaro/ui": ["libs/yaro-ui/src/index.ts"]
    }
  }
}
2
Estilos globales en angular.json
Agrega yaro-ui.scss antes que styles.scss para que los tokens CSS estén disponibles cuando carguen los estilos de la aplicación.
angular.json
{
  "projects": {
    "mi-app": {
      "architect": {
        "build": {
          "options": {
            "styles": [
              "libs/yaro-ui/src/styles/yaro-ui.scss", // ← primero
              "src/styles.scss"
            ]
          }
        }
      }
    }
  }
}
3
Google Fonts en index.html
YARO usa Outfit (principal) y JetBrains Mono (numérico/código). Agrega este link en el <head> de tu aplicación.
src/index.html
<head>
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
  <link
    href="https://fonts.googleapis.com/css2?family=Outfit:wght@300;400;500;600;700&family=JetBrains+Mono:wght@400;500&display=swap"
    rel="stylesheet"
  >
</head>
4
Importar componentes en tu feature
Todos los componentes son standalone. Impórtalos directamente en el array imports del componente que los use. No hay módulo que registrar.
mi-feature.component.ts
import {
  YaroButtonComponent,
  YaroBadgeComponent,
  YaroInputComponent,
  YaroCardComponent,
  YaroKpiComponent,
} from '@yaro/ui';

@Component({
  selector: 'app-mi-feature',
  standalone: true,
  imports: [
    YaroButtonComponent,
    YaroBadgeComponent,
    YaroInputComponent,
    YaroCardComponent,
    YaroKpiComponent,
  ],
  template: `
    <yaro-card title="Ventas hoy">
      <yaro-kpi label="Total" value="$1.240.000" [delta]="8.3" />
    </yaro-card>
    <yaro-button variant="primary">Guardar</yaro-button>
  `,
})
export class MiFeatureComponent {}
