Moléculas

Combinaciones de átomos que forman patrones de UI completos. Cada molécula tiene un propósito específico y puede componer organismos más complejos. Todos importan sus átomos internamente — no necesitas declararlos dos veces.

YaroCard yaro-card

Contenedor de superficie con título opcional y slot para acciones. Base de la mayoría de los paneles del sistema. Controla el padding y la sombra con inputs.

Ventas del día Activo
$1.240.000
47 transacciones
Sin sombra · padding sm
Contenido de la card con padding reducido.
// Card básica con título
<yaro-card title="Ventas del día">
  <p>Contenido libre aquí</p>
</yaro-card>

// Con slot de acciones (esquina superior derecha del título)
<yaro-card title="Productos">
  <p>Lista de productos</p>
  <ng-container card-actions>
    <yaro-badge intent="green">Activo</yaro-badge>
  </ng-container>
</yaro-card>

// Con footer de acciones (pie de la card)
<yaro-card title="Confirmar" padding="md" shadow="lg">
  <p>¿Deseas eliminar este registro?</p>
  <ng-container card-footer>
    <yaro-button variant="secondary">Cancelar</yaro-button>
    <yaro-button variant="primary">Confirmar</yaro-button>
  </ng-container>
</yaro-card>
API — @Inputs
PropiedadTipoDefaultDescripción
titlestring''Título del encabezado. Si está vacío, no se renderiza el header
padding'none' | 'sm' | 'md' | 'lg''md'Padding interno del cuerpo de la card
shadow'none' | 'sm' | 'md' | 'lg''md'Nivel de sombra de la card
Slots de contenido
SlotDescripción
defaultContenido principal del cuerpo
card-actionsAcciones en el encabezado, junto al título (badges, botones pequeños)
card-footerPie de la card con borde superior (botones de confirmación)
YaroKpi yaro-kpi

Tarjeta de indicador clave de desempeño. Combina internamente YaroDelta y YaroBadge. Ideal para dashboards y resúmenes de métricas.

Ventas del día $1.24M
↑ +8.3% vs ayer
Meta ✓
Devoluciones 12
↑ +3 vs semana pasada
Revisar
Mesas ocupadas 13/20
65% ocupación 
// KPI básico
<yaro-kpi
  label="Ventas del día"
  value="$1.240.000"
/>

// Con delta y comparación
<yaro-kpi
  label="Ventas del día"
  value="$1.240.000"
  [delta]="8.3"
  compareLabel="vs ayer"
/>

// Con badge de estado
<yaro-kpi
  label="Ventas del día"
  value="$1.240.000"
  [delta]="8.3"
  compareLabel="vs ayer"
  badge="Meta ✓"
  badgeIntent="green"
/>

// Delta invertido (más = peor, ej: devoluciones)
<yaro-kpi
  label="Devoluciones"
  value="12"
  [delta]="3"
  [deltaInverted]="true"
  compareLabel="vs semana pasada"
  badge="Revisar"
  badgeIntent="amber"
/>
API — @Inputs
PropiedadTipoDefaultDescripción
labelstring''Nombre del indicador (encima del valor)
valuestring''Valor principal del KPI (ya formateado)
deltanumberundefinedVariación numérica. Si no se pasa, no se muestra el delta
deltaInvertedbooleanfalseInvierte colores del delta (para métricas donde menos es mejor)
compareLabelstring''Texto de comparación junto al delta (ej: "vs ayer")
badgestring''Texto del badge de estado. Si está vacío, no se muestra
badgeIntentstring'accent'Intent del badge (mismos valores que YaroBadge)
YaroStatBar yaro-stat-bar

Barra de progreso con etiqueta y porcentaje. Ideal para mostrar ocupación, avance de metas o uso de recursos. El color se controla con intent.

Ocupación de mesas 65%
Meta de ventas 82%
Capacidad de bodega 91%
Errores del día 34% 
// Básico
<yaro-stat-bar
  label="Ocupación de mesas"
  [value]="mesasOcupadas"
  [max]="totalMesas"
  intent="green"
/>

// Sin porcentaje visible
<yaro-stat-bar
  label="Meta de ventas"
  [value]="ventasHoy"
  [max]="metaDiaria"
  intent="accent"
  [showPercent]="false"
/>

// Color dinámico según el valor
<yaro-stat-bar
  label="Capacidad"
  [value]="usado"
  [max]="100"
  [intent]="usado > 90 ? 'red' : usado > 70 ? 'amber' : 'green'"
/>
API — @Inputs
PropiedadTipoDefaultDescripción
labelstring''Etiqueta encima de la barra
valuenumber0Valor actual (acepta signal)
maxnumber100Valor máximo para calcular el porcentaje
intent'green' | 'amber' | 'red' | 'blue' | 'accent''accent'Color de la barra de relleno
showPercentbooleantrueMuestra el porcentaje calculado junto al label
YaroStockBar yaro-stock-bar

Indicador de nivel de inventario. Calcula automáticamente el nivel (ok / low / critical) y cambia de color sin configuración adicional. Los umbrales son ajustables.

Arroz Diana 500g 80 / 100
OK
Aceite Girasol 1L 22 / 100
Stock bajo
Sal Refisal 500g 5 / 100
Crítico 
// Básico — nivel automático (ok / low / critical)
<yaro-stock-bar [current]="producto.stock" [max]="producto.stockMax" />

// Umbrales personalizados
<yaro-stock-bar
  [current]="producto.stock"
  [max]="producto.stockMax"
  [lowThreshold]="40"
  [criticalThreshold]="15"
/>

// En una tabla de inventario
@for (item of inventario; track item.id) {
  <tr>
    <td>{{ item.nombre }}</td>
    <td><yaro-stock-bar [current]="item.stock" [max]="item.max" /></td>
  </tr>
}
API — @Inputs
PropiedadTipoDefaultDescripción
currentnumber0Cantidad actual en inventario (acepta signal)
maxnumber100Cantidad máxima de referencia
lowThresholdnumber30Porcentaje a partir del cual el nivel es "low" (ámbar)
criticalThresholdnumber10Porcentaje a partir del cual el nivel es "critical" (rojo)
YaroNavItem yaro-nav-item

Elemento de navegación con icono, label y badge opcional. Usado internamente por YaroSidenav, pero también puede usarse de forma independiente en menús personalizados.

🏠 Dashboard
💰 Ventas
📦 Inventario 3
👥 Clientes
⚙️ Configuración 
// Básico
<yaro-nav-item label="Dashboard" icon="🏠" />

// Activo (seleccionado)
<yaro-nav-item label="Ventas" icon="💰" [active]="true" />

// Con badge de notificación
<yaro-nav-item
  label="Inventario"
  icon="📦"
  badge="3"
  badgeIntent="red"
  (itemClick)="navigate('inventario')"
/>

// Dinámico — ruta activa
@for (item of navItems; track item.id) {
  <yaro-nav-item
    [label]="item.label"
    [icon]="item.icon"
    [active]="currentRoute === item.id"
    [badge]="item.badge"
    (itemClick)="navigate(item.id)"
  />
}
API — @Inputs / @Outputs
PropiedadTipoDefaultDescripción
@Inputlabelstring''Texto del item de navegación
@Inputiconstring''Emoji o carácter de icono
@InputactivebooleanfalseAplica el estilo de ítem seleccionado
@Inputbadgestring''Texto del badge. Si está vacío, no se renderiza
@InputbadgeIntentstring'accent'Intent del badge de notificación
@OutputitemClickEventEmitter<void>—Emite al hacer clic en el item
YaroFormRow yaro-form-row

Fila de formulario con grid responsivo. Agrupa campos bajo un título de sección con soporte para 1, 2 o 3 columnas. Colapsa a 1 columna automáticamente en pantallas menores a 600px.

Datos del cliente *
Información básica de identificación
Dirección de entrega
// 2 columnas con título requerido
<yaro-form-row
  label="Datos del cliente"
  hint="Información básica de identificación"
  [cols]="2"
  [required]="true"
>
  <yaro-input label="Nombre"   [(ngModel)]="nombre"   />
  <yaro-input label="Apellido" [(ngModel)]="apellido" />
</yaro-form-row>

// 3 columnas
<yaro-form-row label="Dirección" [cols]="3">
  <yaro-input label="Ciudad"        [(ngModel)]="ciudad"   />
  <yaro-input label="Barrio"         [(ngModel)]="barrio"   />
  <yaro-input label="Código postal"  [(ngModel)]="codPostal" />
</yaro-form-row>
API — @Inputs
PropiedadTipoDefaultDescripción
labelstring''Título de la sección del formulario
hintstring''Texto de ayuda debajo del título
cols1 | 2 | 31Número de columnas del grid. Colapsa a 1 col bajo 600px
requiredbooleanfalseMuestra asterisco rojo junto al título
